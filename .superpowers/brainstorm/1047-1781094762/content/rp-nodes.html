3 个新增 RP 节点 — 端口与配置定义

全部注册在 rp-core 插件（node.plugin.json），executor 实现在 executor.mjs

1. rpInputParser — 玩家输入解析

Node: rpInputParser
Categoryroleplay
Color#d97706
Inputtext (user_input) — 玩家原始输入
Outputs parsed (json) — 结构化解析结果
{ speech, action, intent, emotion, entities[], triggers[] }
Config parseRules (textarea) — 解析指令
language (select: zh/en)

2. rpContextAssembler — 上下文组装

Node: rpContextAssembler
Categoryroleplay
Color#d97706
Inputs parsed (json) — 来自 rpInputParser
character (character_profile) — 角色卡
scene (scene_state) — 场景状态
worldbook (context) — 世界书命中
memory (context) — 记忆召回
Output context (context) — 组装后的完整 RP 上下文
Config assemblyTemplate (textarea) — 组装模板
maxTokens (number, 500-4000)

3. rpMemoryWrite — 记忆写入候选

Node: rpMemoryWrite
Categorymemory
Color#d97706
Inputs reply (draft) — 导演输出
notes (analysis) — 连续性检查结果
parsed (json) — 玩家输入解析（可选）
state (scene_state) — 场景状态（可选）
Output candidates (json) — 建议写入的记忆条目列表
[{type, title, content, tags, priority}]
type: relationship | preference | promise | lore | hook
Config autoWrite (boolean, default: false) — 是否自动写入
maxCandidates (number, 1-10, default: 5)
memoryTypes (multiselect) — 启用哪些类型
